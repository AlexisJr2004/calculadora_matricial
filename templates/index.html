Expresiones Matemáticas

Referencia de formatos:

Escribe Verás Ejemplo
sqrt(x) √(x) sqrt(x^2+1) → √(x²+1)
exp(x) eˣ exp(2*x) → e²ˣ
pi π 2*pi*r → 2πr
x^y o x**y xʸ x^2 → x²
sin(x) sin(x) sin(2*pi*x) → sin(2πx)
log(x) log(x) log(100) → log(100)
ln(x) ln(x) ln(e^2) → ln(e²)

Ejemplos prácticos:

  • 2*x^2 + 3*x - 5 → 2x² + 3x - 5
  • sin(x) + cos(2*x) → sin(x) + cos(2x)
  • exp(x) + ln(x+1) → eˣ + ln(x+1)
  • sin(sqrt(x^2 + y^2)) → sin(√(x² + y²))
